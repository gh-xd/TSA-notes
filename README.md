# Introduction

入门

- 时间序列分析
- 时间序列预测
    - 采用历史数据的模型，预测未来观察结果
- 时间序列的组成部分
    1. **等级**：序列时直线，基线值
    2. **趋势**：序列随时间变化，线性增加或减少
    3. **季节性**：序列随时间推移，有重复模式或者行为循环
    4. **噪音**：无法解释的观测值
